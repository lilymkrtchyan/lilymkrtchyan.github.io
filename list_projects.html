Hardware Projects

Explore my projects that involve hands-on hardware design, circuit building, and prototyping.

3D Printed Mechanical Automaton - Spider

3D Printed mechanical automaton of a spider using 22 moving parts designed using Fusion 360 and 3D printed by myself.

Lidar

Creating a "Lidar" project using an IR Range Sensor and Stepper Motor. Data visualized by java-based software SONAR.

Magnetometer + LCD Display

LIS3MDL provides 3D magnetic field data. Communicate with a magnetometer via SPI protocol. Show the readings on the LCD.

Temperature Sensor + LCD Display

Build an LCD thermometer. Read the temperature from the TC74. Display the temperature on LCD display. Communicate via I2C protocol.

Animated Steam Engine

Bidirectional Motor Control. Using the tracking sensor to sense the rotation of the wheel. Change the direction after the wheel rotates for one circle each time. Pause for 5 seconds in between.

Advanced IO

Using the MPU6050 and the LCD display, build a level that moves a dot back and forth on the screen based on one axis of the accelerometer.

Locker

It uses a rotary encoder to register the passcode to the locker. It utilizes sound and LED light to indicate if the passcode was correct or not. After the correct passcode is inputted, the stepper motor opens the locker.

Jack in the Box

Laser cut the box and the clown. The clown jumps out when the photoresistor sensor registers that the lights are off. The clown moves upward supported by a 3D printed gear and pinion system attached to a servo motor. The user can heat the clown head as a game. The LCD display shows the target heats required and how many heats the user has done so far.
